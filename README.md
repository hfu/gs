## How to use

### Install dependencies

#### Ubuntu

```
sudo apt install make jq curl unzip gdal-bin
```

```
npm i -g osmtogeojson
```

#### macOS

```
brew install make jq curl unzip gdal
```

```
npm i -g osmtogeojson
```

### Make

```
make
```
